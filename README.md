# binoxxo-webapp-seed
Binoxxo Single Page Webapp using Seed.

See webapp at: https://msuesskraut.github.io/binoxxo/

# Build & run

Build and run with cargo-make:

    cargo install cargo make
    cargo make build
    cargo make serve
